# Pig-Game
The project is a **Pig Game** implemented using HTML, CSS, and JavaScript. The game involves two players who take turns rolling a dice and accumulating points. The first player to reach 100 points wins the game. The HTML file contains the structure of the webpage, including the layout of the game, the players' scores, and the buttons for starting a new game, rolling the dice, and holding the current score. The CSS file is responsible for styling the webpage, including the layout of the game board, the font, the colors, and the button design. The JavaScript file implements the game's logic, including rolling the dice, adding the current score to the player's total score, changing turns, and declaring the winner. The project demonstrates proficiency in front-end web development and problem-solving skills through the implementation of game logic.
